O Projeto

O projeto surgiu a partir de algo que todos os membros têm em comum, os jogos eletrônicos. A partir disso dado a proposta de trabalho de conclusão de módulo, resolvemos nos basear numa plataforma de games no modelo "streaming play ", como acontece no Playstation Now, Google Stadia e NVIDIA Shield. Assim surgiu a ideia do "Gameflix", uma plataforma com interface simples e intuitiva, onde o usuário teria a facilidade de jogar em qualquer lugar sem se preocupar com os requisitos mínimos para executar o game, podendo jogar em qualquer dispositivo com acesso à internet. Através dessa ideia lúdica foi criada toda interface de navegação do Gameflix para Desktop O layout foi baseado a partir de dois serviços Online: o serviço de streaming "Netflix" e a loja de jogos "Battle Net, da Blizzard" a fim de obter uma melhor UX/UI.
